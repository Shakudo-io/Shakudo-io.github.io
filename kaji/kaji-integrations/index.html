Integrations

Kaji connects to your organization's tools and services through MCP integrations (Model Context Protocol). This means Kaji can read, write, and act on data across your entire tool stack.

What are MCP integrations?​

MCP (Model Context Protocol) is the standard way Kaji connects to external tools and data sources. Each integration provides Kaji with specific capabilities — like reading a database, creating tasks, or sending messages.

You don't need to configure integrations yourself. They are set up by your platform administrator and are automatically available in your Kaji sessions.

Available integrations​

Task Management​

ToolWhat Kaji can do
ClickUpCreate, update, and search tasks, lists, and spaces

Documents and Knowledge​

ToolWhat Kaji can do
NotionRead and update pages, databases, and wikis

Data and Analytics​

ToolWhat Kaji can do
DremioRun SQL queries against your data lake
SupabaseQuery and manage PostgreSQL databases
Neo4jQuery graph databases and explore relationships

Communication​

ToolWhat Kaji can do
MattermostSend messages, summarize threads, and interact with channels

Content Creation​

ToolWhat Kaji can do
GammaGenerate presentations and slides

Memory and Knowledge Graph​

ToolWhat Kaji can do
Graphiti MemoryStore and retrieve facts across conversations using a knowledge graph

Search and Research​

ToolWhat Kaji can do
Context7Search and retrieve contextual information
grep.appSearch across open-source code repositories
MarkItDownConvert documents to markdown for processing

Other​

ToolWhat Kaji can do
NanobananaGenerate and edit images

Using integrations in conversations​

You can mention specific tools in your requests, or let Kaji choose the right integration automatically:

Create a ClickUp task for each bug in this list
Query Dremio for last quarter's revenue by region
Summarize the key points from this Notion page

Kaji will automatically select the appropriate integration based on your request. If you want to use a specific tool, just name it.

Adding new integrations​

New MCP integrations can be added by your platform administrator. If you need access to a tool that isn't listed here, contact your Shakudo administrator.
